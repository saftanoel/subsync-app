# 🔄 SubSync

> **Course Assignment:** MPP (Systems/Software Engineering)
> **Status:** 🚧 Work in Progress (Frontend Phase)

SubSync is a modern web application designed to help users manage their monthly and annual subscriptions, track expenses, and view insightful charts. Currently, the project is in its **frontend and UI/UX development phase**, utilizing mock data and local state. Upcoming assignments will introduce the backend API and database integration.

## ✨ Current Features (Phase 1)

The application currently satisfies the "Silver" and "Gold" frontend requirements:

- **CRUD Operations:** Users can add, view, update, and delete subscriptions.
- **Master-to-Detail Navigation:** Clicking on a subscription row opens a detailed view modal.
- **Dynamic Dashboard:** Charts and total expenses update automatically in the background when tabular data changes.
- **Responsive Design:** Fully optimized for mobile viewports (tested on simulated Pixel 5).
- **Theming:** Persistent Dark Mode (default) and Light Mode (Off-White) with smooth glassmorphism transitions and custom loading screens.
- **User Tracking:** System monitors user activity and sorting preferences using cookies/local storage.

## 🛠️ Tech Stack

- **Framework:** React 18 + Vite
- **Styling:** Tailwind CSS + Shadcn UI
- **Animations:** Framer Motion
- **Routing:** React Router DOM
- **Testing (E2E):** Playwright
- **Testing (Unit/Component):** Vitest + React Testing Library
